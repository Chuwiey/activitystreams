Default 500 (Server Error) middleware

If an error is thrown in a policy or controller, Sails will respond using this default error handler

This middleware can also be invoked manually from a controller or policy: res.serverError( [errors] )

Parameters:

  • errors can be an Array, an Object, or a String.
    (optional errors)
